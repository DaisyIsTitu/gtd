## 🔄 인터랙션 플로우

### 드래그앤드롭
1. **사이드바 → 캘린더**
   - Todo를 드래그하여 원하는 시간에 배치
   - 드래그 중 가능한 시간 슬롯 하이라이트
   - 충돌 시 빨간 테두리 표시

2. **캘린더 내 이동**
   - 기존 Todo 블록 드래그로 시간/날짜 변경
   - 30분 단위 자동 스냅
   - 실시간 충돌 감지

3. **크기 조정**
   - Todo 블록 하단 모서리 드래그
   - 30분 단위로 소요시간 조정
   - 최소 30분, 최대 8시간

### 미리보기 모드 인터랙션
- 미리보기 중에도 드래그앤드롭 가능
- 수정사항 실시간 반영
- [적용] 클릭 전까지 변경사항 미적용

### 호버 효과
- Todo 블록: 상세정보 툴팁 표시
- 빈 시간 슬롯: "+" 아이콘 표시
- 사이드바 Todo: 배경색 변경 및 그림자 효과

### 클릭 인터랙션
- Todo 블록 클릭: 상세 모달 열기
- 빈 시간 클릭: 빠른 추가 모달
- 상태 버튼: 즉시 상태 변경

## 📱 반응형 디자인 브레이크포인트

### 데스크톱 (1920px 이상)
- 사이드바: 300px 고정 (왼쪽)
- 캘린더: 100% - 300px (사이드바 제외)
- 시간 블록: 80px 높이
- 요일 칸: 균등 분할

### 노트북 (1024px - 1919px)
- 사이드바: 280px (약간 축소)
- 캘린더: 100% - 280px
- 시간 블록: 70px 높이

### 태블릿 (768px - 1023px)
- 캘린더: 전체 너비
- 사이드바: 슬라이드 메뉴 (왼쪽에서 나타남)
- 햄버거 메뉴 버튼 표시
- 터치 제스처 지원

### 모바일 (768px 미만)
- 일일 뷰로 자동 전환
- 하단 네비게이션 바
- 풀스크린 모달

### Todo 블록 내 표시 정보
```
┌─────────────────┐
│ 📘 보고서 작성   │ <- 아이콘으로 카테고리 구분
```

### 분할된 Todo 표시
```

```

### 상태별 시각적 구분
- **대기중**: 사이드바에만 표시
- **배치됨**: 일반 색상, 실선 테두리
- **진행중**: 진한 색상, 애니메이션 효과 (pulse)
- **놓친 할 일**: 주황색, 경고 아이콘
- **완료**: 연한 색상, 체크 아이콘, 취소선
